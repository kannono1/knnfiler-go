## dev
```
- Go
go version go1.12.5


- ProjectDir
~/go/src/github.com/kannono1/knnfiler-go/


```

## init
```
go get -u github.com/kannono1/knnfiler-go
go get -u github.com/nsf/termbox-go
go get -u github.com/otiai10/copy
```

## run
```
go run knnfiler.go
```

## build
```
go build -o knnfiler knnfiler.go
```
